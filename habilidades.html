Stack utilizada

Este projeto utiliza HTML, CSS e JavaScript no frontend, além de PHP e MySQL no backend para o formulário de contato e a área administrativa de gerenciamento de mensagens.

HTML

  • Estrutura semântica das páginas (header, main, section, footer).
  • Formulário de contato com campos de texto, e-mail e textarea.
  • Tabela para listagem de contatos na área administrativa.
  • Links de navegação entre as páginas do portfólio.

CSS

  • Uso de variáveis CSS para paleta de cores e tipografia.
  • Layout responsivo com flexbox e media queries.
  • Estilização de cards (cases, formulário, área admin).
  • Efeitos de hover, sombras, bordas arredondadas e gradientes.

JavaScript

  • Destaque automático do link de navegação da página atual.
  • Envio assíncrono do formulário de contato via fetch para o backend em PHP.
  • Animação de entrada dos cards de cases com IntersectionObserver.
  • CRUD de contatos na área administrativa (listar, editar e excluir).
